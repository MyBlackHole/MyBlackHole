# pipx

推荐 [[uv]]

隔离的 python 软件安装环境
每一个都独立

## install
```shell
sudo apt-get install pipx
```

## 使用

- 安装
```shell
pipx install mackup
```

- 重装所有
```shell
pipx reinstall-all
```

- 直接运行
```shell
pipx run https://gist.githubusercontent.com/cs01/fa721a17a326e551ede048c5088f9e0f/raw/6bdfbb6e9c1132b1c38fdd2f195d4a24c540c324/pipx-demo.py
pipx is working!
```
