# hyprland

- 多屏显示
```shell
lvim ~/.config/hypr/hyprland.conf
monitor=HDMI-A-1, 1920x1080, 0x0, 1
monitor=eDP-1, 1920x1080, 1920x0, 1
```

- 设置透明度
```shell
lvim ~/.config/hypr/hyprland.conf
opacity=0.8
```

- 设置壁纸
```shell
lvim ~/.config/hypr/hyprland.conf
wallpaper=/path/to/wallpaper.png
```

- 设置字体
```shell
lvim ~/.config/hypr/hyprland.conf
font=Noto Sans CJK SC, 14
```
