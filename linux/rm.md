# rm

## 选项

|                 |                                                   |
| --------------- | ------------------------------------------------- |
| -d              | 直接把欲删除的目录的硬连接数据删除成0，删除该目录 |
| -f              | 强制删除文件或目录                                |
| -i              | 删除已有文件或目录之前先询问用户                  |
| -r或-R          | 递归处理，将指定目录下的所有文件与子目录一并处理  |
| --preserve-root | 不对根目录进行递归操作                            |
| -v              | 显示指令的详细执行过程                            |

## 例子

|   |   |
|---|---|
|find . -name __pycache__ \|xargs rm -rf|从当前目录递归删除所有同名目录|
