# mtr

## 格式
mtr [-hvrctglspni46] [--help] [--version] [--report] 
[--report-cycles=COUNT] [--curses] [--gtk] 
[--raw] [--split] [--no-dns] [--address interface] 
[--psize=bytes/-s bytes] 
[--interval=SECONDS] HOSTNAME [PACKETSIZE]

mtr -h 提供帮助命令 
mtr -v 显示mtr的版本信息 
mtr -r 已报告模式显示 

其中-c的说明是：–report-cycles COUNT 

第三列:是显示的每个对应IP的丢包率 

第四列:显示的最近一次的返回时延 

第五列:是平均值 这个应该是发送ping包的平均时延 

第六列:是最好或者说时延最短的 

第七列:是最差或者说时延最常的 

第八列:是标准偏差
