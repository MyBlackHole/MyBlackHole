# dmesg
系统日记

## 参数
```shell
-c  #显示信息后，清除ring buffer中的内容
-s<缓冲区大小>  #默认值为8196，刚好等于ring buffer的大小
-n  #设置记录信息的层级
-D  #禁用打印消息到控制台
-E  #启用打印消息到控制台
-h  #打印帮助文本并退出
-k  #打印内核消息
-n  #设置将消息记录到控制台的级别
-r  #打印原始消息缓冲区
-s  #使用多少大小的缓冲区来查询内核环缓冲区。 默认情况下为16392
-T  #打印人类可读时间戳
-t  #不打印内核的时间戳
-u  #打印用户空间消息
-V  #输出版本信息并退出
-x  #将设施和级别（优先级）编号解码为可读的前缀
```

## 例子
- 持续打印
```shell
sudo dmesg -w
```
