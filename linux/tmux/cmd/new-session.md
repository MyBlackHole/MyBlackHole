# new-session

新建会话

```
-A -- 附加到现有会话（如果它已经存在）
-c -- 指定会话的工作目录
-d -- 不将新会话附加到当前终端
-D -- 与 -A 一起使用，分离附加到会话的其他客户端
-E -- 不应用更新环境选项
-e -- 指定环境变量
-f -- 指定客户端标志
-F -- 指定输出格式
-n -- 指定初始窗口名称
-P -- 创建新会话后打印有关它的信息
-s -- 命名会话
-t -- 指定目标会话
-x -- 指定宽度
-X -- 与 -D 一起使用，将 SIGHUP 发送到所连接客户端的父级
-y -- 指定高度
```


- 创建新会话
```shell
<!-- -d 后台 -->
<!-- -s 命名 -->
tmux new-session -s ssh -d
```
