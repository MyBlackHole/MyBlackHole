# hexedit

十六进制编辑器

```shell
使用：hexedit [-s | --sector] [-m | --maximize] [-h | --help] [文件名]
OPTIONS
    -s， - sector格式化显示以包含整个扇区。
    -m， - maximimize尝试最大化显示。
    -h， - help显示用法。

快捷键
    <，>：转到文件的开头/结尾
    右：下一个角色
    左：前一个角色
    下：下一行
    上一篇：上一行
    主页：行首
    结束：行结束
    PUp：向前翻页
    PDown：向后翻页
杂项
    F2：保存
    F3：加载文件
    F1：帮助
    Ctrl-L：重写
    Ctrl-Z：暂停
    Ctrl-X：保存并退出
    Ctrl-C：退出而不保存
    tab：切换十六进制/ ascii
    return：go to
    backspace：撤消上一个字符
    Ctrl-U：全部撤消
    Ctrl-S：向前搜索
    Ctrl-R：向后搜索
    剪切和粘贴
    Ctrl-Space：设置标记
    Esc-W：复制
    Ctrl-Y：粘贴
    Esc-Y：粘贴到文件中
```
