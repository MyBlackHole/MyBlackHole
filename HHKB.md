# HHKB Studio 备忘录

连接方式切换
有线连接：fn + ctrl + 0
蓝牙设备连接：
蓝牙设备 1：fn + ctrl + 1
蓝牙设备 2：fn + ctrl + 2
蓝牙设备 3：fn + ctrl + 3
蓝牙设备 4：fn + ctrl + 4
蓝牙配对信息管理
进入配对模式
按键操作：fn + q
存放配对设备
将即将配对的设备存放到指定位置：
位置 1：fn + ctrl + 1
位置 2：fn + ctrl + 2
位置 3：fn + ctrl + 3
位置 4：fn + ctrl + 4
删除配对信息
删除所有配对信息：
按键操作：fn + q（进入等待配对模式）** **fn + z + delete
删除个别配对信息：
按键操作：fn + q（进入等待配对模式）** **fn + ctrl + delete + [1~4]（指定位置）
取消配对
按键操作：fn + q（进入等待配对模式）** **fn + x
模式切换（仅无线状态下有效）
切换为 Win 模式：fn + ctrl + w
切换为 Mac 模式：fn + ctrl + m
